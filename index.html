VPN Server

A. Pengertian VPN adalah singkatan dari “Virtual Private Network”, merupakan suatu koneksi antara satu jaringan dengan jaringan lain secara pribadi melalui jaringan Internet (publik). Disebut dengan Virtual Network karena VPN menggunakan jaringan Internet sebagai media perantaranya alias koneksinya bukan secara langsung. Dan disebut Private Network karena VPN bersifat pribadi maksudnya hanya orang tertentu saja yang dapat mengaksesnya.
C. Fungsi dari VPN Berikut adalah fungsi dari VPN: 1. Kerahasiaan (Confidentially) 2. Keutuhan data (Data Integrity) 3. Autentikasi sumber (Origin Authentication)
D. Manfaat dari VPN 1. Remote Access 2. Keamanan 3. Dapat menghemat biaya setup jaringan
Contoh Aplikasi VPN Server 1. ExpresVPN 2. NordVPN 3. CyberGhost
